## 🔧 如果遇到问题

### 问题 1: 页面显示空白或 404

**解决**:
1. 检查前端路由配置
2. 确认访问的 URL 正确
3. 查看浏览器控制台是否有错误

### 问题 2: API 请求失败（401 未授权）

**解决**:
1. 重新登录
2. 检查 localStorage 中是否有 `admin_token`
3. 打开浏览器开发者工具 → Application → Local Storage

### 问题 3: 创建/编辑失败

**解决**:
1. 打开浏览器控制台查看错误信息
2. 检查必填字段是否都已填写
3. 查看后端日志：`tail -f /tmp/eu-doc-server.log`

### 问题 4: 企业下拉框为空

**解决**:
1. 检查是否有企业数据：访问 http://localhost:3007/api/companies
2. 如果没有，先创建企业
