# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [2.8.1] - 2026-07-21

### Added
- 企业资料审核闭环：平台管理员可预览、通过或拒绝企业上传的资料
- 审核状态与意见在企业端显示
- 审核结果通知上传者和企业管理角色

### Fixed
- 优化管理员审核和账号工作流

## [2.8.0] - 2026-07-21

### Added
- 平台管理真实工作流：用户查询、权限处理、禁用和恢复操作
- 举报处理接入真实 API 和处理流程
- 分类管理支持搜索、新增分类、新增子类和编辑操作
- 系统设置接入真实读取、保存和审计记录

### Security
- 补齐企业资源写接口权限隔离

## [2.7.2] - 2026-07-18

### Security
- 批量上传审计与公开资料安全加固
- 补齐批量导入权限校验

## [2.7.1] - 2026-07-18

### Security
- 生产环境不再暴露密码重置令牌
- 密码变更后旧 JWT 失效
- 补齐公司、产品图片、文档、旧版兼容证书接口和标签关联的企业权限校验
- 服务器隔离数据库安全测试 8 项全部通过

## [2.7.0] - 2026-07-12

### Added
- 平台管理员可跨公司只读查看员工
- 新增全平台操作记录
- 公司级和平台级操作记录默认展示全部时间

## [2.6.1] - 2026-07-12

### Added
- 支持认证前企业申请人角色
- 企业申请人显示并拥有认证前成员管理权限
- 认证通过后自动转为企业拥有者

## [2.6.0] - 2026-07-12

### Added
- 员工权限页接入真实企业成员数据
- 支持添加已注册成员、修改角色和移除成员
- 公司操作记录接入真实审计日志
- 补齐成员变更日志、筛选和 CSV 导出
