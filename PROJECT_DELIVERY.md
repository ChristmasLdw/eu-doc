## 🎓 使用指南

### 首次部署
1. 阅读 `DEPLOYMENT_GUIDE.md`
2. 配置环境变量
3. 构建前端：`npm run build`
4. 启动后端：`cd server && node index.cjs`
5. 配置 Web 服务器（Nginx）
6. 修改默认密码

### 日常使用
1. **用户注册**: 访问 `/admin/register`
2. **管理后台**: 访问 `/admin/login`
3. **企业认证**: 登录后访问 `/company-verification`
4. **文档上传**: 后台 → 产品管理 → 上传文档

### 管理员操作
1. **审核认证**: 后台 → 企业认证审核
2. **查看日志**: 后台 → 操作日志
3. **管理用户**: 后台 → 用户管理
4. **查看确认记录**: 后台 → 上传确认记录

## 🆘 故障排查

### 常见问题

**Q: 后端启动失败？**
```bash
# 检查端口占用
lsof -i :3007
# 检查数据库文件
ls -la server/data/
```

**Q: 前端构建失败？**
```bash
# 清理缓存
rm -rf node_modules dist
npm install
npm run build
```

**Q: API 请求 404？**
- 检查 Nginx 配置
- 检查后端是否运行
- 检查 API_BASE_URL 配置

**Q: 邮件发送失败？**
- 检查 SMTP 配置
- 开发环境会返回 token
- 查看后端日志
