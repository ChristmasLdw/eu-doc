# EU Doc - 文档工具

## 项目简介
文档管理和展示工具。

## 技术栈
- 构建工具：Vite 或 Webpack（待确认）
- Node.js 后端

## 本地运行
```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建
npm run build
```

## 部署
```bash
# 构建后部署 dist 目录
npm run build
# 将 dist/ 复制到服务器
```

## 修改指南
- `src/` - 源代码
- `dist/` - 构建产物
- `server/` - 后端服务

## 注意事项
- 修改后需要重新构建
- 确保生产环境配置正确
