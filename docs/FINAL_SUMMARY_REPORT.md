### 3. 问题修复 (7个)

#### 问题1: 多语言只切换导航栏
- **症状**: 点击语言按钮，只有导航栏文字改变
- **原因**: 其他页面未使用 `useTranslation` hook
- **解决**: 在所有页面添加 `t()` 函数调用
- **文件**: 9 个页面文件

#### 问题2: 搜索框主题不适配
- **症状**: 搜索框固定深色背景
- **原因**: CSS 使用硬编码颜色
- **解决**: 替换为 `var(--bg-primary)`
- **文件**: HomePage.module.css, SearchPage.module.css

#### 问题3: 搜索页空白
- **症状**: 访问搜索页显示白屏
- **原因**: 缺少函数导入
- **解决**: 导入 `getSortOptions` 等函数
- **文件**: SearchPage.jsx

#### 问题4: 证书数据不显示
- **症状**: 数据库有 48 条，前端显示为空
- **原因**: Vite 代理未配置 `/eu-doc/api`
- **解决**: 添加代理配置
- **文件**: vite.config.js

#### 问题5: 遗漏的翻译
- **症状**: 部分文字未翻译
- **原因**: 初次实现遗漏
- **解决**: 补充翻译键值对
- **文件**: zh.json, en.json

#### 问题6: 搜索页主题问题
- **症状**: 搜索结果页主题不适配
- **原因**: 建议下拉框使用硬编码颜色
- **解决**: 替换为 CSS 变量
- **文件**: SearchPage.module.css

#### 问题7: 缩略图加载失败 ⭐
- **症状**: 显示"图片加载失败"
- **原因**: Vite 代理未配置静态文件路径
- **解决**: 添加 `/eu-doc/certificates` 代理
- **文件**: vite.config.js
- **影响**: 29 张缩略图现可正常显示
