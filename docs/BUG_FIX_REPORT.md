### 2. 主题切换问题 ✅

#### 问题描述
- 主题切换没有把搜索框的颜色切换过来
- 搜索建议下拉框使用了硬编码的深色背景色 `#141b2d`
- 多处使用了 `rgba()` 硬编码颜色，不适配主题

#### 修复内容
- ✅ **首页搜索框 (HomePage.module.css)**
  - 修复搜索建议下拉框背景：`#141b2d` → `var(--bg-card)`
  - 修复 hover 背景：`rgba(255, 255, 255, 0.1)` → `var(--bg-card-hover)`
  - 修复标题渐变：硬编码渐变 → `var(--text-primary)`
  
- ✅ **搜索建议样式优化**
  - 移除所有 `rgba()` 硬编码颜色
  - 使用主题变量：`var(--accent-blue)`, `var(--status-active)` 等
  - 优化不同类型建议的视觉区分（使用左边框代替半透明背景）
  - 标签颜色统一使用主题色 + 白色文字

- ✅ **公司卡片样式**
  - 修复 hover 阴影：`rgba(0, 0, 0, 0.1)` → `var(--shadow-card)`
  - 修复图标渐变：硬编码渐变 → `var(--accent-gradient)`

#### 测试验证
```
1. 打开首页，点击搜索框
2. 输入关键词，显示搜索建议下拉
3. 切换到暗黑模式 🌙
4. 验证搜索框、建议下拉都是深色背景
5. 切换到明亮模式 ☀️
6. 验证搜索框、建议下拉都是浅色背景
7. 所有颜色应该平滑过渡
```
