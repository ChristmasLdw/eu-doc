## 🎯 第13个任务详解

### 管理后台Logo上传界面

**实现内容：**
- 在CompaniesPage编辑模态框中集成FileUpload组件
- 仅在编辑企业时显示（新增时不显示）
- 显示当前Logo预览
- 支持JPG/PNG格式，最大2MB
- 上传成功后自动刷新企业列表
- 添加友好的提示文案

**技术实现：**
```jsx
{editingCompany && (
  <div className={styles.formField}>
    <label>{t('admin.companiesPage.companyLogo')}</label>
    <FileUpload
      accept=".jpg,.jpeg,.png"
      maxSize={2 * 1024 * 1024}
      onUpload={(file) => handleLogoUpload(editingCompany.id, file)}
      existingFile={editingCompany.logoUrl ? { 
        url: editingCompany.logoUrl, 
        type: 'image' 
      } : null}
    />
    <p className={styles.fieldHint}>
      {t('admin.companiesPage.logoHint')}
    </p>
  </div>
)}
```

**用户流程：**
1. 管理员进入企业管理页面
2. 点击"编辑"按钮
3. 在编辑模态框中看到Logo上传区域
4. 拖拽或点击上传Logo图片
5. 查看实时上传进度
6. 上传成功后看到新Logo预览
