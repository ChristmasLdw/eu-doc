#### 3. v2.0 新 API 开发 (Phase 4)
**完成内容**:
- ✅ 产品管理API（6个端点）
- ✅ 文档管理API（5个端点）
- ✅ 分类管理API（5个端点）

**API 体系**:
```
产品 API
├── GET    /api/v2/products                产品列表
├── GET    /api/v2/products/:id            产品详情
├── GET    /api/v2/products/:id/documents  产品文档
├── POST   /api/v2/products                创建产品
├── PUT    /api/v2/products/:id            更新产品
└── DELETE /api/v2/products/:id            删除产品

文档 API
├── GET    /api/v2/documents               文档列表
├── GET    /api/v2/documents/:id           文档详情
├── POST   /api/v2/documents               创建文档
├── PUT    /api/v2/documents/:id           更新文档
└── DELETE /api/v2/documents/:id           删除文档

分类 API
├── GET    /api/v2/categories              分类列表
├── GET    /api/v2/categories?tree=true    树形结构
├── GET    /api/v2/categories/:id          分类详情
├── POST   /api/v2/categories              创建分类
├── PUT    /api/v2/categories/:id          更新分类
└── DELETE /api/v2/categories/:id          删除分类

标签 API
├── GET    /api/v2/tags                    标签列表
├── GET    /api/v2/tags/:id                标签详情
├── POST   /api/v2/tags                    创建标签
├── PUT    /api/v2/tags/:id                更新标签
├── DELETE /api/v2/tags/:id                删除标签
├── POST   /api/v2/tags/:id/products       给产品打标签
├── DELETE /api/v2/tags/:id/products/:pid  移除产品标签
├── POST   /api/v2/tags/:id/documents      给文档打标签
└── DELETE /api/v2/tags/:id/documents/:did 移除文档标签
```
