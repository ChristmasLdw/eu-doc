## 技术实现亮点

### 搜索算法
```javascript
// 匹配度评分系统
if (valueLower.startsWith(q)) {
  matchScore = 3; // 开头匹配
} else if (valueLower.includes(' ' + q) || valueLower.includes('-' + q)) {
  matchScore = 2; // 单词边界
} else {
  matchScore = 1; // 普通包含
}
```

### 响应式设计标准
- **触摸目标：** 最小44px × 44px（iOS标准）
- **字体大小：** 移动端输入框16px（防止缩放）
- **断点策略：** 768px（平板）、640px（大手机）、480px（小手机）
- **下拉高度：** 最大60vh（避免遮挡）
