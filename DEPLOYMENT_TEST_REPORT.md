## 🐛 已知问题

### 警告级别

1. **JS Bundle 过大**
   - **问题**: 主 JS 文件 593KB（gzip 后 171.9KB）
   - **影响**: 首次加载较慢
   - **建议**: 代码分割、懒加载
   - **优先级**: 中

2. **SMTP 未配置**
   - **问题**: 开发环境未配置真实 SMTP
   - **影响**: 邮件验证功能返回 token，需手动处理
   - **建议**: 生产环境配置 SMTP 服务
   - **优先级**: 高（生产环境必须）

### 待完善功能

1. **企业切换组件未集成**
   - **状态**: 组件已创建，需集成到导航栏
   - **优先级**: 中

2. **单元测试缺失**
   - **状态**: 无单元测试
   - **建议**: 添加 Jest + React Testing Library
   - **优先级**: 中
