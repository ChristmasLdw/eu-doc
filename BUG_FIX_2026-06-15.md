## 🐛 修复的问题

### 1. 搜索框下拉栏透明度问题 ✅

**问题描述**：
在黑夜模式下，搜索框下拉建议栏的透明度过低，导致背景内容穿透，影响文字阅读体验。

**问题截图**：用户反馈截图显示下拉框背景半透明，背景元素清晰可见

**根本原因**：
- `.suggestions` 样式中缺少足够的背景不透明度
- 黑夜模式下没有特殊的背景处理

**修复方案**：
```css
/* src/pages/SearchPage.module.css */

.suggestions {
  /* 原有样式... */
  
  /* 新增：增加背景模糊和不透明度 */
  backdrop-filter: blur(20px);
  -webkit-backdrop-filter: blur(20px);
}

/* 暗黑模式下增强下拉框背景不透明度 */
[data-theme="dark"] .suggestions {
  background: rgba(30, 41, 59, 0.98); /* 98% 不透明度 */
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.5);
}
```

**效果**：
- ✅ 黑夜模式下下拉框背景几乎完全不透明（98%）
- ✅ 添加毛玻璃效果（backdrop-filter）
- ✅ 增强阴影效果，提升层次感
- ✅ 文字清晰可读，不受背景干扰

## 🔍 技术细节

### CSS 暗黑模式选择器
使用 `[data-theme="dark"]` 属性选择器来针对暗黑模式添加特殊样式：
```css
[data-theme="dark"] .suggestions {
  background: rgba(30, 41, 59, 0.98);
}
```

### 翻译键的容错处理
使用 `||` 运算符提供后备方案：
```javascript
{t(`search.categories.${cat}`) || cat}
```
这样即使翻译键不存在，也会显示原始值。

### backdrop-filter 浏览器兼容性
- 添加 `-webkit-` 前缀支持 Safari
- 现代浏览器都已支持此特性
