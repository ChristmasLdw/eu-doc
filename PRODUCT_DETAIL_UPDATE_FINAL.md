### 3. 数据来源

**产品信息**：
```javascript
GET /eu-doc/api/v2/products/:id          // 产品详情
```

**文档列表**：
```javascript
GET /eu-doc/api/v2/products/:id/documents // 产品所有文档
```

**证书元数据**（额外获取）：
```javascript
GET /eu-doc/api/certificates/:id         // 证书完整信息
// 包含：standard, issuer, issueDate, expiryDate, category, model 等
```

**文档类型映射**：
- `certificate` → 资质证书（47 个）
- `declaration_of_conformity` → DoC 声明文件（0 个）
- `manual` → 使用说明书（0 个）
